# Smart Health Analytics System

An AI-powered multi-disease prediction system that analyzes patient health parameters and predicts the likelihood of several diseases using machine learning models.

## Features

- Diabetes Prediction
- Heart Disease Prediction
- Breast Cancer Detection
- Parkinson's Disease Prediction
- Liver Disease Prediction

## Tech Stack

- Python
- Scikit-learn
- XGBoost
- Streamlit
- Pandas
- NumPy
- Matplotlib

## How It Works

1. User enters medical parameters.
2. The system processes the data using trained ML models.
3. The model predicts the probability of disease.
4. The result is displayed through an interactive Streamlit dashboard.
